# 03. 策略波段管理 (Positions)

记录流水后，我们需要将流水打包升华为**策略波段 (Position)**，以此核算单次投资周期的盈亏。在 CryptoFolio 中，一个 Position 代表您针对某一个币种的一次完整狙击（建仓 -> 平仓）。

## 构建一个新波段

在 **Positions** 页面点击 **New Position**：
- 为其选择对应的 `Symbol (如 SOL/USDT)`，并设置一个开端日期。
- 可选地填写 `Name (命名该策略)` 以及备忘录 `Notes` 存放您的思考逻辑。
- 随后它会作为一个 `OPEN (进行中)` 的卡片存在。

## 将散单流水链接进波段 (Linking Transactions)

进入刚才创建的 Position 详情页：
1. 这里分为上下两大板块：上方的 **核心结算区** 和 下方的 **交易账本**。
2. 页面中间会显示 **Available Transactions (此交易对名下目前闲置、未关联的流水)**。
3. 点击任意闲置流水右侧的 **Link (+)** 按钮，即可将它“装”进这台波段计算器中。

## 智能持仓引擎与多空双向支持

- 当您往里面塞入第一笔流水时，CryptoFolio 会**自动判定方向**：
  - 第一笔是买入(`BUY`)：系统判定该波段为 **做多 (LONG)**。
  - 第一笔是卖出(`SELL`)：系统判定为您借币抛售，即 **做空 (SHORT)**。
- 随着您逐渐将买单和卖单 `Link` 进去，顶部的核心报表面板会实时动态推演：
  - **Your Holdings**: 您现在还持有的净头寸数量（支持小数点级别的超高精度）。
  - **Average Entry Price**: 您的加权平均建仓成本价。
  - **Realized PnL**: （已平仓部分的）落袋利润。
  - **Unrealized PnL**: （尚未平仓部分的）根据市场现价浮动的账面盈利。

> **提示**：假如您在中途通过 `Unlink (-)` 按钮踢出了某笔错误关联的交易，所有报表、方向、持仓均价也会瞬时重新推演！

## 结束与归档

当一次波段的盈利达到您的目标，或者扫到止损单后：
1. 确保与之相关的平仓卖出流水（或买空补回流水）都被 `Link` 完了。
2. Holdings（持有量）应归零，或剩下极少的粉尘。
3. 点击详情页右上角的 **Close Position**。
该波段的历史表现、胜负将正式冻结并录入您的战绩库，纳入仪表盘的全局胜率统计当中。
