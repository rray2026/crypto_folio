# 05. Data Security & Personalization (Security & UI)

The **Settings** page carries your data lifeline and visual experience.

## Data Backup (The Lifeline)

Since CryptoFolio uses a **Purely Local Storage (Privacy First)** mode, your data is only stored in the IndexedDB of that browser.
- **Export**: One-click generation of a `.json` full snapshot. Contains: transaction flows, position logic, strategy notes, and pinned lists.
- **Restore**: **Warning:** Importing will overwrite all current browser data; please ensure you have backed up the previous version.

## System Preferences

- **Pinned Pairs (Market Pinned)**: 
  - Set the currencies you care about most (e.g., `BTC/USDT`).
  - These currencies will be prioritized in the **Dashboard** and will be pinned in the dropdown list when recording transactions.
- **Theme**: 
  - Manually switch between `Dark` / `Light` or automatically change following `System`.

---

> [!TIP]
> It is recommended to perform a manual backup and store it in your cloud drive or offline USB drive after each large-scale import or monthly review.
