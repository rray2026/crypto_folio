# 02. The Art of Transaction Logging (Transactions)

Accurately tracking position gains is inseparable from the precise recording of each basic **Transaction**. This is the bottom-layer fuel that keeps the CryptoFolio engine running.

## Recording a Single Transaction

Click **Add -> Manual Entry** in the upper right corner:
- **Type**: Choose `BUY` or `SELL`.
- **Symbol**: Trading pair, such as `BTC/USDT`.
- **Price & Amount**: Enter the average transaction price and quantity.
- **Fee**: **Important:** Please record the手续费 (fee) generated by the transaction, as it will directly affect your net profit settlement.

## Binance Bulk Import

CryptoFolio is deeply optimized for Binance transaction export files:

1. **Smart Deduplication**: The system automatically intercepts imported transactions through the Binance Order ID. You can export the full statement at any time and click import without worrying about doubling the data.
2. **Fee Aggregation**: The main order row exported by Binance often does not show the fee; the fee is scattered in the sub-transaction rows below. Our parsing engine will automatically traverse all sub-rows and aggregate the most accurate total fee for you.
3. **Price Fallback**: If the average transaction price data is missing, the engine will automatically fall back to using the delegated price for estimation, ensuring that the data is not left blank.

## Association Status Tracking

- **[Linked] Tag**: Transactions in the list that have been associated with a position will show a blue tag.
- **Locking Mechanism**: Linked transactions are not allowed to change the `Symbol` (to prevent position logic from collapsing), but notes or time can be modified.
- **Multiple Associations**: A transaction can be assigned to multiple positions (i.e., "splitting" logic). Click the tag to check which strategies the transaction has flowed to.

---

> [!WARNING]
> Deleting a linked transaction will cause all corresponding positions to be recalculated. If the transaction is the only opening order for a position, the direction of the position will be reset.
