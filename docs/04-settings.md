# 04. 系统设置与数据迁移 (Settings)

**Settings (设置)** 页面不仅关乎您的视觉偏好，还承载着您资产数据的绝对安全保障机制。

## 个性化与预设

- **Predefined Trading Pairs (自选交易对)**: 
  - 您可以预设长线追踪的标的（如 `BTC/USDT`, `PEPE/USDT`）。
  - 这些交易对此后将在您整个应用的下拉菜单中享有置顶推荐权，免去手打的麻烦。
  - 列表还会展现由 Binance API 抓回来的极速最新现价，甚至可以随时点底部的强制刷新按钮强制同步一次。
  
- **Appearance (外观主题)**: 
  - 支持随心所欲切换 `Light (白天)`、`Dark (暗黑)`，或追随您的操作系统的模式 (`System`)。

## 数据备份与还原 (Data Backup & Restore)

由于应用完全运行在本地浏览器中，您的每次重装系统/更换浏览器都会导致数据留空。**请务必养成定期导出的好习惯！**

1. **导出一份冷备份 (Export Backup)**:
   - 点击导出，系统会将您的数百条交易、精心打磨好的波段体系、乃至目前的黑暗主题偏好**整体融合打包**。
   - 这会调用浏览器下载一封格式为 `cryptofolio-backup-[日均].json` 的微型文件至您的硬盘。这封文件非常轻量级。

2. **恢复我的档案 (Import Data)**:
   - 在新电脑或浏览器中打开全新的 CryptoFolio，来到此页面，点击导入，并选择您的那封 `JSON` 文件。
   - **注意！** 导入过程极具破坏性。它会彻底抹除您当前浏览器这残存的空壳数据表，并将备份文件里的资产数据以毫秒级速度一比一复刻落库，最后自动刷新网页还您当年江山！
